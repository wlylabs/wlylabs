### 🚀 About Me

I'm an independent **solo builder** who turns ideas into shipped products — fast.
I work the **vibe coding** way: intuition-driven design, tight iteration loops,
and AI doing the heavy lifting so I can focus on *what* to build instead of just
*how* to type it out.

No team. No bureaucracy. Just me, an idea, and a clean commit history.

### 🧠 How I Build

- 🎯 **Idea first** — I start from the problem, never the boilerplate
- ⚡ **Ship fast** — small, frequent iterations over big-bang releases
- 🤖 **AI-native** — I build end to end alongside AI coding tools
- ✨ **Keep it clean** — simple, readable, maintainable by design
